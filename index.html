Natural Language Forms

Natural Language forms can be a great way to create an intuitive UI. Add the class .natural-language to your <form> tag and Stachestrap works it's magic.

Note that natural-language currently supports regular text <input> tags. This includes most types. Inputs with type="file" rely on stachestrap.js for styling, as well as an additional<span> element in your html markup. It supports <select> tags but requires an additional class of .nl-select to be added.

Cards

Cards are a great way to display information, taking full advantage of Bootstrap's grid system. They appear nicely on light-grey backgrounds, a design pattern you may have seen on Google, Facebook, LinkedIn and other popular sites (circa 2013/2014). Create a div within a column with the class .card. Use any of the following to add color to your card's background:

  • .card-white or .card-default
  • .card-black
  • .card-primary
  • .card-success
  • .card-info
  • .card-warning
  • .card-danger

Wells

Stachestrap's well classes are just a very simple way to add colorful backgrounds to your typical Bootstrap well. Include the standard .well class on your div, but extend it futher with:

  • .well-white or .well-default
  • .well-black
  • .well-primary
  • .well-success
  • .well-info
  • .well-warning
  • .well-danger
